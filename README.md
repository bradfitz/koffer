# Der verloren Koffer

A German-language mini-game adventure: help Larry chase his lost suitcase (and
its diamond) across Germany — land the plane, find the bag, drive the train,
match the clothes, dodge the darts, whack the smugglers.

## ▶️ [Play it here](https://bradfitz.github.io/koffer/js/)

Originally a Java applet; the playable version is a faithful JavaScript port
(see [js/README.md](js/README.md), including the Quirks section).

## History

* **Spring 1997**: [@bradfitz](https://github.com/bradfitz) and
  [@dabernards](https://github.com/dabernards) make this Java applet game for
  their high school German project
* **July 2026**: AI ports it to JavaScript to play in modern browsers
